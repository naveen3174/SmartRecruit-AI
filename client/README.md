# 🌐 SmartRecruit AI - Frontend

This is the React-based frontend for the SmartRecruit AI platform, built with **Vite** for high performance.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Dev Server**:
   ```bash
   npm run dev
   ```

## 🛠 Features Included
- **Framer Motion**: For premium SaaS-level animations and transitions.
- **Tailwind CSS**: Modern UI styling with glassmorphism effects.
- **Recharts**: Data visualization for interview scores.
- **Lucide React**: Clean and consistent iconography.
- **React Webcam**: For capturing interview recordings.

## 📱 Mobile Preview
Run `npm run dev --host` and check the **Network URL** on your mobile device (must be on the same Wi-Fi).
